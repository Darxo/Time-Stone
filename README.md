# Description

Time Stone is a quality of life mod that combines and improves the features existing auto-pause and speed-up mods.

# List of all Features

## World Speed

- Make all three Vanilla speed states configurable
- Allow adjusting of speed during escort missions
- Allow adjusting of speed during camping

## Auto Pause

- Add settings to cause any of the following events to pause the game while on the world map
	- Discover Hostile Party (on)
	- Discover Hostile Camp (off)
	- Discover Unique Location (off)
	- Sunrise Near Hostile Party (off)
	- Nightfall Near Hostile Party (off)
	- Sunrise Near Attackable Location (off)
	- Nightfall Near Attackable Location (off)
	- Sunrise Near Allied Settlement (off)
	- Leave Event/Contract Screen (on)
- Add configurable Unpause-Protection that prevents accidental unpauses after an Auto-Pause triggered (500ms)
- Forget world parties after not seeing them for a configurable amount of ingame hours (2 hours), allowing Auto-Pause to trigger again on them
- Add setting to trigger the discover sound for hostile parties when you meet them again (on)
- Add setting to play a new unique sound effect when discovering any unique location (on)

## Tactical Speed

- Add settings to configure two tactical speed settings
- Add two customizable keybinds to swap between the two tactical speed settings (F2 and F3)

# Why this mod over other similar mods?

## Unique about this mod

- 2-in-1: You only need to install one mod, instead of speed and pause mods
- Configurable Vanilla World Speed
- More Auto-Pause Events (than **Configurable Pause**)
- Discover sound for unique location
- Configurable World Speed
- 2 Tactical Speed settings you can configure and switch between via hotkeys
- Small pause-button animation when Auto-Pause triggers

## Downsides of this mod

### Missing Features

- Pause on Arrival (**Swifter**, **Mount n Blade Pause**)
- Unpause when choosing new destination (**Swifter**, **Mount n Blade Pause**)
- Pause on sunrise while camping (**Swifter**)
- Pause on nightfall while camping (**Swifter**)
- Pause on Escort over city (**Swifter**)
- 4th World Speed State (**Swifter**)

### General

- Requires **MSU** and **Modern Hooks** which makes it a bit more fragile should those mods break
- This mod is more likely to break from a Vanilla update, because of its size, compared to a simple speed-up mod like **Quicker**
- Just like **Swifter** and **Quicker**, this mod speeds up combat speed by changing the virtual clock. This approach is known to rarely cause glitches or crashes in combat

# Compatibility

- Is safe to remove from- and add to any existing savegames
